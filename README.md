# 🏪 THE MARKET - Marketplace Website

A professional, responsive marketplace application connecting businesses and customers.

## Features

### 🔐 Authentication

- **User Registration**: Customers and business owners can create accounts
- **Secure Login**: Role-based login system (Customer/Business)
- **Session Management**: Local storage-based authentication

### 👥 Customer Features

- Browse all available products
- **Advanced Search**: Search by product name, business name, or category
- **Smart Sorting**: Sort by name, price (low-high, high-low), or newest
- View detailed product information in modal
- Responsive product grid layout

### 🏢 Business Features

- **Product Management**: Add, edit, and delete products
- **Inventory Control**: Manage product stock
- **Product Details**: Include images, prices, descriptions, and categories
- **Dashboard**: View all your products in a table format
- **Search**: Filter your own products

## Getting Started

1. **Extract/Open** the project in your browser
2. **Navigate to** `index.html` to start
3. **Create an account** using the Register button
4. **Login** with your credentials

## User Types

### Customer Account

1. Register as a Customer
2. Login to access the product marketplace
3. Search and sort products
4. View product details

### Business Account

1. Register as a Business Owner
2. Enter your business name and category
3. Login to access your dashboard
4. Add products with images, prices, and descriptions
5. Edit or delete products as needed
